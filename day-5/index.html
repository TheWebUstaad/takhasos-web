Day 5

Today we will learn about the for attributes.

The for attribute is used in conjunction with the label element to associate a label with a form control.

When a user clicks on the label, the associated form control will receive focus.

  • Click on the label to focus the input field.
  • Type something in the input field.
  • Click the button to submit the form.
